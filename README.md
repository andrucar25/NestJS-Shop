# Shop API

1. Clone proyect
2. ```pnpm install```
3. Clone file ```.env.template``` y rename to ```.env```
4. Change envs
5. Up database
```
docker-compose up -d
```
6. Execute SEED 
```
http://localhost:3000/api/seed

7. Run project: ```pnpm start:dev```
